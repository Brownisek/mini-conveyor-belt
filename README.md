# mini-conveyor-belt
### A mostly 3D-printable mini conveyor belt made from very cheap and accessible components.

The conveyor is still in development, so expect frequent changes to the models and code.


## Parts List

- **Stepper motor (28BYJ-48)** – ~$2  
  https://share.temu.com/TmuSf8Q3nBB

- **Stepper motor driver (ULN2003)** – ~$1  
  https://share.temu.com/TmuSf8Q3nBB

- **Ball bearings (5×16×5 mm)** – ~$3 for 10 pcs  
  https://share.temu.com/1PeBttO7vWB

- **Flexible belt material**
  I used a rubber tourniquet, but any thin, flexible rubber strap should work.

Files and Images [here](https://github.com/Brownisek/mini-conveyor-belt/tree/Conveyor-files/conveyor_files)
